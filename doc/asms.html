<static> indexTypeValuesFromCollection(collection) → {Dictionary}

Utility method that generates a JSON dictionary mapping simple TypeValue identifiers to object TypeValue equivalents based on the contents of the passed in Activity Streams Collection object. This is most useful when initializing reusable libraries of verb and objectType definitions and using those in conjunction with the typeValueResolver.
