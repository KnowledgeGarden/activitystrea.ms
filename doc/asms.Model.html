Class: Model

asms. Model

<abstract> new Model(a)

A Model is an object with zero or more OwnPropertyDescriptors. They are used to dynamically define the properties of the various Activity Stream objects. This allows us to easily mutate the properties of things like ActivityObject, Activity, Collection, etc based on properties such as objectType, extensions, etc.

Model objects cannot be created directly. Rather, they are generated by extending existing models provided by the asms.Models namespace
